# Enum ERouteResultType
## 描述
<font class="since">开始于 : v0.0.6</font>  
路由结果类型  
## 枚举值
#### CHAIN
路由链,和redirect不同，浏览器地址不会改变  
##### 初始值
'chain'  
#### JSON
json数据,默认类型  
##### 初始值
'json'  
#### NONE
什么都不做  
##### 初始值
'none'  
#### REDIRECT
重定向  
##### 初始值
'redirect'  
#### STREAM
文件流，主要用于文件下载  
##### 初始值
'stream'
